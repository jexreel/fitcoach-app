# FitCoach 💪

App de ejercicio personalizada según tu **peso, talla y meta**.

## 🚀 Demo en vivo
👉 [Abrir FitCoach](https://jexreel.github.io/fitcoach-app)

## ✨ Funcionalidades
- Programa semanal personalizado por objetivo (perder peso, ganar músculo, tonificar, resistencia, flexibilidad)
- Animaciones en tiempo real de cada ejercicio (Canvas 2D)
- Descripción paso a paso de cada movimiento
- Temporizador de descanso con vibración
- Seguimiento de progreso y logros
- PWA instalable en Android (sin tienda de apps)

## 📱 Instalar en Android
1. Abre la URL en Chrome para Android
2. Toca "Añadir a pantalla de inicio"
3. ¡Listo! Se instala como app nativa

## 🛠 Tecnologías
- HTML5 + CSS3 + JavaScript puro
- Canvas 2D para animaciones
- Service Worker (offline support)
- PWA Manifest
- localStorage para persistencia
